## FAQ

**1) What does “custom install” change?**  
It makes you set the security and privacy defaults (Kill Switch, startup behavior, protocol choices) during setup instead of after.

**2) Does fast vpn support AES-256?**  
Yes—AES-256 encryption is part of the standard security posture in this setup.

**3) Is a Kill Switch required?**  
If you care about leak prevention, yes. Turn it on before your first real session.

**4) How do I pick the fastest server?**  
Start with the closest region, then try a few nearby **Global Servers** and keep the best-performing one for your network.
